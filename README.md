AI-based flood risk prediction system using real-time weather data.

## Features
- Machine Learning model for flood prediction
- Real-time weather data (OpenWeatherMap API)
- Cloud deployment ready (AWS EC2)
- SMS alert system support
- Disaster early warning system

## Technologies Used
- Python
- Scikit-learn
- Pandas & NumPy
- OpenWeatherMap API
- AWS EC2
- Twilio (SMS alerts)

## How it Works
1. Load training dataset
2. Train AI model
3. Fetch live weather data
4. Predict flood risk
5. Send alerts if risk is high
